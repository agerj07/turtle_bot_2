# turtle_bot_2

Guide TurtleBot2 to predefined waypoints navigating a partially known environment.
Developed ROS-based occupancy grid mapping, EKF localization, A* path planning, and pure pursuit control.
Navigated to all 10 preset waypoints without collision.
